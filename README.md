# addressbook
Address Book is a place to store contacts


<b>Frontend:</b> 
<ul>
  <li>Angularjs for mv* framework
  <li>Bootstrap for CSS components
</ul>

<b>Backend:</b> 
<ul>
  <li>Spring MVC for REST endpoints
  <li>JPA/Hiberate as ORM framework 
  <li>Spring framework
  <li>JUNIT
</ul>
  
<b>Database: </b> 
<ul>
  <li>SQL Server 2012
</ul>

<b>IDE used:</b> 
<ul>
  <li>Eclipse
  <li>Sql Developer
</ul>
  
<b>HTTP server:</b> 
<ul>
  <li>Tomcat 7.0.67
</ul>
  
<b>Build tool:</b> 
<ul>
 <li> Maven
</ul>
